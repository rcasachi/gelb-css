CSS Reset

GelbCSS Reset was built upon Normalize, providing an initial and standardized structure for all browsers. All changes that we've done in reset file from Normalize was described below:

  • All elements (including *::before and *::after were globally set a box-sizing: border-box and margin-top: 0.
  • No font-size was declared on the reset file, except by body default as 16px.
  • The <body> sets as global font-family, line-height, background-color, color, and text-align.
  • It was set list-style: none for all nav ul.
  • It was set quote: none for all blockquote, q and content: ''; content: none for all blockquote:before, blockquote:after, q:before, q:after.
  • It was set border-collapse: collapse; border-spacing: 0 for <table> element.
