Typography

We list below the full documentation of GelbCSS typography.

Native font stack

GelbCSS uses a native font stack based on Medium and Bootstrap +4 to select the best font-family for each OS and device.


		font-family: 
			-apple-system, // Safari for OS X and iOS (San Francisco)
			BlinkMacSystemFont, // Chrome < 56 for OS X (San Francisco)
			"Segoe UI", // Windows
			"Roboto", // Android
			"Oxygen", // KDE
			"Ubuntu", // Ubuntu
			"Cantarell", // GNOME
			"Fira Sans", // Firefox OS
			"Droid Sans", // older versions of Android
			"Helvetica Neue", "Arial", sans-serif; // Basic web fallback
